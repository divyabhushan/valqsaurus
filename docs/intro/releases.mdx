## V2.0.4 ( Mar 30, 2021 )

#### New Features

* <Link to={useBaseUrl('storage/admin-console#valq-sign-in-methods')}>ValQ Sign-in Methods</Link>: ValQ allows 2-way sign-in for the users. Users can log-in with Native username/password managed by ValQ apart from Microsoft Office 365 sign-in.

#### Enhancements

* In the Plan tab, the <Link to={useBaseUrl('planning/time_series')}>AutoForecast</Link> feature freezes the **Source Periods**. The period values selected for the **Target Period** will be displayed for the Source Periods as well.

#### Bug Fixes



## V2.0.0 ( Jan 20, 2021 )

#### New Features

1. <Link to={useBaseUrl('storage/user-security')}>User Security</Link> feature in ValQ Collaborate to grant specific access to the Model/nodes, Scenarios, and Planning Series to user(s) and group(s).
1. <Link to={useBaseUrl('analysis/sensitivity-analysis')}>Sensitivity Analysis</Link> of the effect of varied input KPIs on a selected target node. You can select the input nodes that are to be simulated and percentage of variance. Together the 2 parameters determine the result on the output node in the form of a Tornado chart.
1. Digital Board room experience with <Link to={useBaseUrl('simulation/touch-support')}>Touch support</Link>. ValQ features are now automatically optimized for Touch Support on touch-enabled devices for both Canvas and Tree view.
1. The <Link to={useBaseUrl('planning/plan_features#7-writeback')}>Plan Tab now includes the **Writeback** feature</Link> that triggers to send data to an external API.
1. You can enable to show the <Link to={useBaseUrl('model/node#show-secondary-kpis-in-table-view')}>Secondary KPIs </Link> in the Table view.

#### Enhancements

1. Planning toolbar has been redesigned and reordered with 2 more features - 'Writeback', and 'Size columns to fit'.
1. Single lock icon acts as a toggle to <Link to={useBaseUrl('planning/plan_features#37-hold-and-release')}>Hold/Release</Link> the cells.
1. Auto fit/sizing of columns width in the Plan tab, Simulation table view, and Variance Analysis tab.

#### Bug Fixes

1. The selections made in the <Link to={useBaseUrl('analysis/variance_analysis')}>Variance Analysis</Link> tab are now saved and retained even when you quit the Analyze tab.
1. The value of Node completely gets diplayed in the <Link to={useBaseUrl('planning/plan_overview')}>Plan Grid (Decimal points)</Link> irrespective of the node's configuration.
1. While selecting the weights the "Tab" button should be used to move to next period and "Shift+Tab" should be used to come back to previous period. This is disabled for 1st(Shift+Tab) and last period(Tab)

## V1.8.3 ( Jul 30,2020 )

#### Features
* Added feature **Rolling Planning** to automatically roll out 12 months forecast over a set time frame.
* Added feature to export your model as a web data source with the option <Link to={useBaseUrl('storage/export')}>Export as datasource</Link>.
* **New Table view** - Table view can now be customized to edit the columns and get summarized views.
* Added the <Link to={useBaseUrl('analysis/attribution-analysis')}>Attribution Analysis</Link> which analysis the key attributes that contribute to the overall result simulation.
* <Link to={useBaseUrl('storage/model-merge')}>Model Merge</Link> enabled for a selected node from the sub-model to another node in the workspace.
  
#### Enhancements
* Made performance based **Member Console changes**
* **Notes** feature is now replaced as **Comments**
* Planning and Allocation history comments now available as consolidated comments.
* Improved the **Writeback** data JSON format
* Enhanced the Node configuration [**Formulas**](https://docs.ValQ.com/docs/faq/formula-list)

## V1.7.0 ( May 12,2020 )

#### Features

1. Added **Model Merge** feature that allows users to merge ValQ models from different users within the workspace.
1. Added **[Time Series Forecasting](https://docs.ValQ.com/docs/planning/time_series) - Auto Forecast** feature.
2. The **Member/ValQ Console** account available for users/groups and models management in your workspace.

#### Enhancements

1. Enabled **Scenario Collaboration** (ValQ - Professional, Enterprise) for Private and Shared Scenarios with view/edit access.
1. Enabled **Read-only Mode** in Power BI Service, with read-only access for *Visualization (Table/Tree mode), Nav Panel, Export, Compare, and Variance Analysis* like features.
1. Added **Model Details** in the workspace, along with details like Series/Scenarios under *Member/ValQ Management Console*.
