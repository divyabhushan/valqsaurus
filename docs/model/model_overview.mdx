_Model screen_

The _Model_ screen has 2 sections; namely:

- **Nodes Panel**: This is the sidebar-panel on the left, to edit the model nodes and build model.
- **Canvas(visual)**: This is the canvas on the right, to visualize the model in tree/table structure.
