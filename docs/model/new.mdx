</div> <br />

<div style={{ textAlign: "justify" }}>

### 2.1 Edit a simple model

<div style={{ textAlign: "justify" }}>
  The nodes on a Simple Model are disabled and you cannot perform any
  configuration on it. But you can generate a child node of the top node and
  proceed to configure it based on your needs.
</div> <br />
