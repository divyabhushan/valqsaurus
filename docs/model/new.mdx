## 2. Create a Simple Model

<div style={{ textAlign: "justify" }}>
To build a Simple Model, you need to import data into Power BI and activate ValQ. Then create a simple model in ValQ.
</div> <br />

<div style={{ textAlign: "justify" }}>
  The steps to create a Simple Model has been covered under{" "}
  <Link to={useBaseUrl("intro/model")}>Using ValQ - An Example</Link> or

    this pdf

  for step by step instructions on how to build a Simple Model. <br />

A Simple model looks like this

<div class="center">
  <Zoom>
    <img alt="Simple model" src={useBaseUrl("/doc-images/model/new4.png")} />
  </Zoom>
  <p>Simple Model</p>

<br />

> The Calculation Method for a Simple Model is always set to Sum Children

### 2.1 Edit a simple model

<div style={{ textAlign: "justify" }}>
The nodes on a Simple Model are disabled, and you cannot perform any configuration on it. But you can generate a child node of the top node and
configure it based on your needs.
</div> <br />

<div class="center">
  <Zoom>
    <img
      alt="Edit a simple model"
      src={useBaseUrl("/doc-images/model/new11.png")}
    />
  </Zoom>
  <p>Edit a Simple Model</p>
</div>

### 2.2 Convert to advanced model

To configure the nodes of a Simple Model, convert the the model to an advanced model by clicking the magic wand icon: 

<div style={{ textAlign: "center" }}>
  <Zoom>
    <img alt="Magic wand" src={useBaseUrl("/doc-images/model/new13.png")} />
  </Zoom>
  <div class="center"><p>Magic Wand</p></div>
</div>

Click **Yes** to confirm.

<div style={{ textAlign: "center" }}>
  <Zoom>
    <img
      alt="Convert to advanced model"
      src={useBaseUrl("/doc-images/model/new14.png")}
    />
  </Zoom>
  <div class="center"><p>Convert to advanced model</p></div>
</div>

You have successfully converted the Simple Model into an Advanced Model. You can now configure all the node levels similar to an Advanced Model.

<div class="center">
  <Zoom>
    <img alt="Advanced model" src={useBaseUrl("/doc-images/model/new15.png")} />
  </Zoom>

## 3. Create an Advanced Model

Use Create an Advanced Model option to build a model on ValQ from scratch manually.

<div style={{ textAlign: "center" }}>
  <Zoom>
    <img
      alt=" Create an Advanced Model"
      src={useBaseUrl("/doc-images/model/new18.png")}
    />
  </Zoom>
</div>

In the below screen; a parent node and two child nodes are added by clicking the **+** button.
The node levels are displayed in the Node Editor pane as shown:

<div class="center">
  <Zoom>
    <img alt="Advanced model" src={useBaseUrl("/doc-images/model/new19.png")} />
  </Zoom>
  <p>Advanced Model</p>
</div>

Now you can go ahead and build the model as per your needs using the Model tab options.
