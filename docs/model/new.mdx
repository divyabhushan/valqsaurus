## 2. Create a Simple Model

<div style={{textAlign: 'justify'}}>
To build a Simple Model, first you need to import data into Power BI and activate ValQ. This is followed by creating a
simple model on ValQ.
</ div> <br />

<div style={{textAlign: 'justify'}}>
The steps to create a Simple Model has been covered under <Link to={useBaseUrl('docs/intro/model')}>Using ValQ - An Example</Link> or you can refer <a href="https://valq.com/wp-content/uploads/valq-for-microsoft-power-bi-beginners-tutorial.pdf" target="_blank">this pdf</a> for step by step instructions on how to build a Simple Model. <br />
</ div> <br />

A Simple model looks like this

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="Simple model" src={useBaseUrl('/doc-images/model/new4.png')} />
  </Zoom>
</ div>

<div style={{textAlign: 'center'}}>

_Simple model_
</div>

> **Note** <br />
The Calculation Method for a Simple Model is always set to Sum Children

### 2.1 Edit a simple model

<div style={{textAlign: 'justify'}}>
The nodes on a Simple Model are disabled and you cannot perform any configuration on it. 
But you can generate a child node of the top node and proceed to configure it based on your needs.
</ div> <br />

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="Edit a simple model" src={useBaseUrl('/doc-images/model/new11.png')} />
  </Zoom>
</ div>

<div style={{textAlign: 'center'}}>

_Edit a simple model_
</div>
