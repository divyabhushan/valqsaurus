<div style={{textAlign: 'justify'}}>
Users who are not part of the ValQ workspace are presented with Read-only mode 
capabilities which allows them to view the model and perform run-time operations without enforcing login.
This feature is beneficial for embedding Power BI reports.
</div> <br />

> **Note** <br />
Read-only mode is _not_ available in Power BI Desktop

## Reading View

Read-only mode is available only in Power BI Service as Reading View which is it's default mode.

<div style={{ textAlign: "left" }}>
  <Zoom>

  </Zoom>
</div><br />

Let's take a look at the ValQ tabs on Reading View of Power BI Service.

### 1. Model

Users need to be part of the ValQ workspace to access Model tab.

### 3. Simulate

<div style={{textAlign: 'justify'}}>

</div> <br />

Users who are part of the workspace can click the Edit Report option on Power BI Service and then login to workspace to get the complete access and use ValQ.
